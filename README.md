# Netflix Content Analysis Case Study

# Project Overview

This project analyzes Netflix's content library to understand trends in movies and TV shows over time. The goal is to explore content growth, release strategies, and key contributors (actors and directors) using exploratory data analysis (EDA).

The analysis focuses on identifying patterns in Netflix’s content expansion, release timing, and production strategies.

# Key Business Insights

1. Netflix significantly increased content production after **2015**.
2. Movies dominate the platform due to **fast content scaling**.
3. TV shows play a major role in **user retention and binge viewing**.
4. Content release timing appears **strategically planned around audience availability**.
5. Data quality gaps exist in metadata fields like **cast and director**.
